# vue-crash-2021

## Project setup

```
npm install
```

### Compiles and runs backend

```
npm run backend
```

Backend available at:
http://localhost:5000/

### Compiles and hot-reloads for development

```
npm run serve
```

Frontend available at:
http://localhost:8080/

### Compiles and minifies for production

```
npm run build
```
